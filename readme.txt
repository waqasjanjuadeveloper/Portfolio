# Muhammad Waqas - Personal Portfolio Website

Welcome to the repository for my personal portfolio website! This project is designed to showcase my skills, experience, and projects as a Native Android & Flutter App Developer. The site features a modern, responsive design with dynamic animations and a premium UI aesthetic.

## 🎨 Key Features & Highlights
- **Responsive Design**: Fully functional and beautifully scaled on desktops, tablets, and smartphones.
- **Dark/Light Mode Toggle**: Built-in theme switcher with glassy aesthetics adapting to both modes seamlessly.
- **Glassmorphism Theme**: Features clean, translucent, frosted-glass effects on the sticky navbar and UI sections.
- **Advanced Animations & Interactions**:
  - CSS-based ambient text-rotation effects on the hero image.
  - Floating element animations for visual depth.
  - Type-writer style text entrance.
- **Dynamic Projects Grid**: Includes an interactive, filterable portfolio gallery to display Web, App Dev, and Design projects.
- **Modern Layouts**: Built optimally using CSS Grid and Flexbox for exact precision in the Spider-web Skills layout, Contact Form, and Footer structure.

## 🛠️ Technology Stack
- **HTML5**: Semantic tags and structure.
- **CSS3**: Custom properties (Variables), Keyframe animations, Grid & Flexbox Layouts, Media queries.
- **Vanilla JavaScript**: For DOM manipulation, theme toggling, and layout functionality.

## 📦 Tools & Libraries Used
- **[Boxicons](https://boxicons.com/)**: For clean, scalable iconography used in the footer, services, and navigation.
- **[MixItUp 3](https://www.kunkalabs.com/mixitup/)**: Enables fluid filtering and sorting animations for the Portfolio/Projects section.
- **[Swiper JS](https://swiperjs.com/)**: For advanced touch slider carousels (integrated via CDN).
- **[ScrollReveal JS](https://scrollrevealjs.org/)**: Implements smooth element appearances as the user scrolls down the page.
